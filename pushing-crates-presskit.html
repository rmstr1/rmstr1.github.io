Description

Pushing Crates is a fun and challenging sliding puzzle game that tests your logic and spatial skills.

Features

  • 100 levels of increasing difficulty
  • Complete the levels in any order
  • For an extra challenge, try to complete each level in the fewest moves possible

About Normal Bytes Studio

Normal Bytes Studio is a one-person independent game development studio based in Graz, Austria. Specializing in pixel art games, I use generative art and pixel aesthetics to create immersive virtual worlds. In addition to game development, I build dedicated tools for asset creation to streamline workflows and expand creative possibilities.
