Description

Pushing Crates is a fun and challenging sliding puzzle game that tests your logic and spatial skills.

Features

  • 100 levels of increasing difficulty
  • Complete the levels in any order
  • For an extra challenge, try to complete each level in the fewest moves possible

About The Developer

Apes of the Screen is a one-person independent game development studio based in Graz, Austria.
